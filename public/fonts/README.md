# DaggerSquare font

Place the DaggerSquare font files here so titles load correctly on all devices (including mobile/Vercel).

## What to do

1. **Download the font** (free, by NÃ©stor Delgado) from one of:
   - https://www.dafont.com/daggersquare.font (OTF)
   - https://www.fontrepo.com/font/3814/daggersquare (TTF)
   - https://www.onlinewebfonts.com/download/eeb2727b7a3e4951c7a43e819a927132 (TTF, WOFF, WOFF2)

2. **Rename or convert so you have:**
   - `DaggerSquare.woff2` (best for web) and/or
   - `DaggerSquare.woff` and/or
   - `DaggerSquare.ttf`

   If you only get `.ttf` or `.otf`, you can use a converter (e.g. cloudconvert.com) to create `.woff2` and `.woff` for better loading on mobile.

3. **Put the file(s) in this folder:**  
   `public/fonts/`  
   So the app can load: `/fonts/DaggerSquare.woff2`, `/fonts/DaggerSquare.woff`, etc.

Once the files are here and you redeploy, the site will use the real DaggerSquare font everywhere.
